Excel & Power BI para Principiantes

Este sitio está diseñado para ayudarte a dominar las herramientas de Excel y Power BI desde cero. Ideal para quienes están comenzando en el mundo del análisis de datos.

📊 Fórmulas más utilizadas en Excel

Excel es una herramienta poderosa para organizar, analizar y visualizar datos. Aquí tienes las fórmulas básicas e intermedias más comunes, agrupadas por categoría:

🧮 Funciones Matemáticas y Aritméticas

  • SUMA: =SUMA(A1:A10)
  • RESTA: =A1-A2
  • MULTIPLICAR: =A1*A2
  • DIVIDIR: =A1/A2
  • PROMEDIO: =PROMEDIO(A1:A10)
  • MAX: =MAX(A1:A10)
  • MIN: =MIN(A1:A10)
  • REDONDEAR: =REDONDEAR(A1,2)
  • ABS: =ABS(A1)
  • POTENCIA: =POTENCIA(A1,2)

🔤 Funciones de Texto

  • CONCATENAR: =CONCATENAR(A1, " ", B1)
  • IZQUIERDA: =IZQUIERDA(A1,5)
  • DERECHA: =DERECHA(A1,3)
  • LARGO: =LARGO(A1)
  • ESPACIOS: =ESPACIOS(A1)
  • MINUSC: =MINUSC(A1)
  • MAYUSC: =MAYUSC(A1)
  • NOMPROPIO: =NOMPROPIO(A1)
  • HALLAR: =HALLAR("texto",A1)
  • REEMPLAZAR: =REEMPLAZAR(A1,1,5,"nuevo")

🧠 Funciones Lógicas

  • SI: =SI(A1>10, "Mayor", "Menor")
  • Y: =Y(A1>10,B1<5)
  • O: =O(A1>10,B1<5)
  • NO: =NO(A1>10)
  • SI.ERROR: =SI.ERROR(A1/B1, "Error")

🔍 Funciones de Búsqueda y Referencia

  • BUSCARV: =BUSCARV("Producto", A2:B10, 2, FALSO)
  • BUSCARH: =BUSCARH("Ventas", A1:D5, 2, FALSO)
  • INDICE: =INDICE(A2:A10,3)
  • COINCIDIR: =COINCIDIR("Juan",A2:A10,0)
  • DESREF: =DESREF(A1,2,1)

📅 Funciones de Fecha y Hora

  • HOY: =HOY()
  • AHORA: =AHORA()
  • DÍA: =DÍA(A1)
  • MES: =MES(A1)
  • AÑO: =AÑO(A1)
  • FECHA: =FECHA(2025,9,15)
  • DIAS: =DIAS(A1,B1)

📊 Funciones Estadísticas Básicas

  • CONTAR: =CONTAR(A1:A10)
  • CONTAR.SI: =CONTAR.SI(A1:A10, ">5")
  • CONTAR.BLANK: =CONTAR.BLANK(A1:A10)
  • FRECUENCIA: =FRECUENCIA(datos,intervalos)

📈 Fórmulas más utilizadas en Power BI (DAX)

Power BI te permite crear dashboards interactivos y realizar análisis avanzados. Estas son algunas de las fórmulas DAX más comunes, organizadas por categoría:

🧮 Funciones de Agregación

  • SUM: SUM(Ventas[Total])
  • SUMX: SUMX(Ventas, Ventas[Precio] * Ventas[Cantidad])
  • AVERAGE: AVERAGE(Ventas[Total])
  • AVERAGEX: AVERAGEX(Ventas, Ventas[Total])
  • MAX / MIN: MAX(Ventas[Total]) / MIN(Ventas[Total])

🔤 Funciones de Texto

  • CONCATENATE: CONCATENATE(Clientes[Nombre], " ", Clientes[Apellido])
  • LEFT / RIGHT: LEFT(Clientes[Nombre], 3) / RIGHT(Clientes[Nombre], 2)
  • LEN: LEN(Clientes[Nombre])
  • UPPER / LOWER: UPPER(Clientes[Nombre]) / LOWER(Clientes[Nombre])
  • REPLACE: REPLACE(Clientes[Nombre], 1, 3, "XXX")

🧠 Funciones Lógicas

  • IF: IF(Ventas[Total] > 5000, "Alto", "Bajo")
  • AND: AND(Ventas[Total] > 1000, Ventas[Cantidad] > 5)
  • OR: OR(Ventas[Región] = "Norte", Ventas[Región] = "Sur")
  • NOT: NOT(Ventas[Cantidad] > 10)
  • IFERROR: IFERROR(Ventas[Total] / Ventas[Cantidad], 0)

🔍 Funciones de Filtro y Contexto

  • CALCULATE: CALCULATE(SUM(Ventas[Total]), Ventas[Región] = "Norte")
  • FILTER: FILTER(Ventas, Ventas[Total] > 1000)
  • ALL: CALCULATE(SUM(Ventas[Total]), ALL(Ventas))
  • ALLEXCEPT: CALCULATE(SUM(Ventas[Total]), ALLEXCEPT(Ventas, Ventas[Región]))
  • VALUES: VALUES(Clientes[Ciudad])

🔗 Funciones de Relación

  • RELATED: RELATED(Clientes[Nombre])
  • RELATEDTABLE: RELATEDTABLE(Ventas)
  • LOOKUPVALUE: LOOKUPVALUE(Clientes[Nombre], Clientes[ID], Ventas[ClienteID])

📅 Funciones de Fecha y Hora

  • TODAY: TODAY()
  • NOW: NOW()
  • YEAR / MONTH / DAY: YEAR(Ventas[Fecha]) / MONTH(Ventas[Fecha]) / DAY(Ventas[Fecha])
  • DATEDIFF: DATEDIFF(Clientes[Registro], TODAY(), DAY)
  • EDATE: EDATE(Ventas[Fecha], 2)

📊 Funciones Estadísticas Básicas

  • COUNT: COUNT(Ventas[ID])
  • COUNTA: COUNTA(Clientes[Nombre])
  • COUNTROWS: COUNTROWS(Ventas)
  • DISTINCTCOUNT: DISTINCTCOUNT(Clientes[Ciudad])
  • RANKX: RANKX(ALL(Clientes), SUM(Ventas[Total]))
